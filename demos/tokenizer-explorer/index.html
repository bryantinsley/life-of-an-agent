Tokenizer: cl100k_base — the BPE used by GPT-4 family. Different models use different tokenizers (Gemini uses SentencePiece with ~256K pieces, shared with Gemma; Llama uses its own SentencePiece BPE), but the family of behaviors — sub-word, leading-space-sensitive, multilingually uneven — is shared across all of them.

Why this matters: the token count is the bill, the latency, the context budget. Bytes do not predict tokens; they have to be measured.
