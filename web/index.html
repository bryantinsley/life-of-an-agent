Life of an Agent 10-session SRE curriculum
how LLMs and agents actually work

From a token in,
to a tool call out.

Ten sessions tracing what happens between the moment text arrives at a model and the moment an agent runs a real action — and what changes when you have to operate the whole loop at scale.

Format 45 min content + 15 min Q&A, weekly · Audience Google SREs, no ML background assumed · Pedagogy interactive demos in every session

Sessions

Each session ships five artifacts: slides, a facilitator guide, a narrative companion document (so the deck stands alone), a fact-check ledger with sources, and a study prompt you can paste into a chatbot between sessions to test understanding.
